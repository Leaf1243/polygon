# 可変正多角形ジェネレーター (Fractional Polygon Generator)

入力された数値（整数・小数）に基づいて、正多角形や星型正多角形（非整数角形）を動的に計算して描画するTkinter製のデスクトップアプリケーションです。

---

## 🌟 特徴

* **小数の入力に対応**: `5.5` などの小数を入力すると、内部で分数（例: `11/2`）に変換し、美しい星型多角形を描画します。
* **直感的な操作**: 数値を入力して「描画」ボタンを押すだけのシンプルなインターフェース。
* **自動クリーンアップ**: 連続して異なる数値を入力しても、キャンバスが自動でクリアされ、新しい図形が再描画されます。

---

## 🛠️ 仕組み

入力された数値 $n$ をPythonの `fractions.Fraction` モジュールを用いて最簡分数 $\frac{p}{q}$ に変換します。
* $p$ : 頂点の総数
* $q$ : 頂点を結ぶ際の間隔（ステップ数）

これにより、通常の正多角形（$q=1$）だけでなく、美しい星型正多角形（$q \ge 2$）の描画が可能になっています。

---

## 🚀 使い方

### 1. 動作環境
